| ------------- | -----:|

| GPIO12     |   D11 |

| GPIO02=D8     | D13=B机方向 |
| GPIO12=D5     |   D11=B机动力 |
| GPIO16=D7 |    D12=A机方向 |
| GPIO13=D4 |    D3=A机动力 |
